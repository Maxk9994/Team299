# Welcome to StackEdit!

To use this project you must have git and visual studio with python installed 
then follow these steps:



In the command line
Clone the repository from the github link

  

     git clone https://github.com/Maxk9994/Team299


Navigate to the folder

    cd calendar app

ensure py is installed

    py --version

If not then troubleshoot online for how to install it can vary
 
Do these two commands

    python -m venv env

env\Scripts\activate

Install flask

    pip install flask 

Finally run the app

    py app.py

This will inform you the local port it is running on paste this into your browser and you can access the app
